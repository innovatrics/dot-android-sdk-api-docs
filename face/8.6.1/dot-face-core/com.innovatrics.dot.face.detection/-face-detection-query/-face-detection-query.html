Parameters

Face image quality query.

evaluatePassiveLiveness

Passive liveness evaluation flag. This method requires dot-face-passive-liveness module. Default value is false.

createFullFrontalImage

Create an ICAO full frontal image of the face. If boundaries of the normalized image leak outside of the original image, a white background is applied. Default value is false.

createTemplate

Create the face template which can be used for matching. This flag requires dot-face-verification module. Default value is false.
