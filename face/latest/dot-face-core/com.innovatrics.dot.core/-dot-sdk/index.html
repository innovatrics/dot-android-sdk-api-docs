fun deinitialize()

Deinitialize DOT SDK. This operation should be called on background thread. If no DOT UI components are processing camera images, deinitialization proceeds without waiting for UI work (note: deinitialization itself is synchronous and may still take some time). If UI components are actively processing images, this method blocks until that processing finishes and then completes deinitialization.
