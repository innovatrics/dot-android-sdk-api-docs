BgrRawImage

data class BgrRawImage(val size: ImageSize, val bytes: ByteArray)

Represents an image with 8-bit BGR color components with the values Blue, Green, and Red stored in 3 bytes.

The 3 bytes of color image pixel:

  • 0th byte: BB - blue

  • 1st byte: GG - green

  • 2nd byte: RR - red
